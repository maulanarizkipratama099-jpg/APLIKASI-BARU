Petunjuk Penggunaan

Arsip Surat: Catat surat masuk dan keluar. Lampirkan link Google Drive jika menyimpan scan surat di Cloud.
Cetak Surat: Buat SK, Surat Keterangan, atau Pengantar secara instan menggunakan form. Kop Resmi & Barcode akan otomatis tercetak di kertas (format Portrait).

Gunakan tab Input Manual untuk menyimpan data 1-per-1. Gunakan Upload Excel untuk memasukkan ratusan data sekaligus (Wajib mendownload template excel yang disediakan terlebih dahulu). Gunakan menu Cetak Filter untuk memisahkan kelas/jenis kelamin siswa atau PNS/Honorer (Cetak Tabel Laporan otomatis format Landscape).

Gunakan fitur ini sebagai pengganti buku fisik harian. Data akan terekam secara real-time ke dalam Spreadsheet dan bisa dicetak kapan saja dibutuhkan.

Isi form Pengaturan agar sistem mengenali Identitas Sekolah Anda untuk keperluan Kop Surat. Klik menu Database di kiri bawah untuk langsung melihat isi file master Spreadsheet di Google Drive.
